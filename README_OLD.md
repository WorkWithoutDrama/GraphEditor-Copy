# Редактор графов с AI ассистентом

Веб-редактор графов с интегрированным AI ассистентом, аналогичным функциональности телеграмм-бота из bot.py.

## Структура проекта

```
.
├── index.html          # Основной HTML файл
├── styles.css          # Стили
├── script.js           # Основной JavaScript редактора графов
├── graph-manager.js    # Класс GraphManager для AI ассистента
├── api.py              # Простой HTTP сервер для API
├── bot.py              # Исходный телеграмм бот
└── README.md           # Эта инструкция
```

## Функциональность

### 1. Редактор графов
- Создание узлов (действия и состояния)
- Создание связей между узлами
- Удаление элементов
- Сохранение/загрузка проектов в JSON

### 2. AI ассистент (правая панель)
- Чат-интерфейс для общения с AI
- Загрузка текстовых файлов
- Генерация моделей систем из описания
- Автоматическое создание графа на основе сгенерированной модели

### 3. API сервер
- `/api/generate-model` - генерация модели из текста
- `/api/health` - проверка работоспособности

## Запуск

### 1. Запуск API сервера
```bash
python api.py
```
Сервер запустится на http://localhost:5000

### 2. Открытие веб-интерфейса
Откройте файл `index.html` в браузере.

### 3. Использование AI ассистента
1. Нажмите кнопку "AI Ассистент" в панели управления
2. В правой панели введите описание системы или загрузите файл
3. AI сгенерирует модель и отобразит граф в основном окне

## Интеграция с bot.py

Функциональность AI ассистента повторяет логику телеграмм-бота:
- Анализ текстового описания системы
- Генерация JSON модели
- Создание графовой визуализации

## Расширение функциональности

Для полной интеграции с оригинальным bot.py:
1. Перенести логику `generate_model_from_text` из bot.py в api.py
2. Добавить поддержку OpenRouter API
3. Реализовать валидацию через Pydantic модели
4. Добавить генерацию графических графов через Graphviz

## Требования

- Python 3.9+
- Браузер с поддержкой JavaScript ES6+
- Для полной функциональности: установить зависимости из `requirements.txt`
