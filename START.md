# Инструкция по запуску Graph Editor

## Вариант 1: Простой запуск (демо-режим)

1. **Просто откройте `index.html` в браузере**
2. Нажмите кнопку **"Graph Manager"**
3. Введите текст или загрузите файл
4. Система будет работать в **демо-режиме** без API сервера

## Вариант 2: Полный запуск с AI API

### Шаг 1: Запустите API сервер
```bash
# В терминале 1
python api.py
```
API сервер запустится на `http://localhost:5000`

### Шаг 2: Запустите веб-сервер для файлов
```bash
# В терминале 2
python serve.py
```
Веб-сервер запустится на `http://localhost:8000` и откроет браузер

### Шаг 3: Используйте Graph Editor
1. Браузер откроется автоматически
2. Нажмите кнопку **"Graph Manager"**
3. Введите описание системы или загрузите файл
4. AI создаст графовую модель

## Решение проблем

### Ошибка CORS
Если видите ошибку CORS:
- Используйте **Вариант 2** (запустите `serve.py`)
- Или используйте **Вариант 1** (демо-режим)

### API недоступен
Если API сервер не запущен:
- Система автоматически перейдет в демо-режим
- Будут создаваться демо-модели на основе ключевых слов

### Граф не обновляется
- Убедитесь, что cytoscape загружен
- Проверьте консоль браузера на ошибки
