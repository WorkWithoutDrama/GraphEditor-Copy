# ДЛЯ ЗАПУСКА GRAPH EDITOR:

## ПРОСТО:
**ДВАЖДЫ КЛИКНИТЕ НА `RUN.command`**

## ИЛИ:
1. Откройте Терминал
2. Перейдите в эту папку:
   ```bash
   cd /путь/к/этой/папке
   ```
3. Запустите:
   ```bash
   ./RUN.command
   ```

## ЧТО БУДЕТ:
1. ✅ Запустится AI сервер
2. ✅ Запустится прокси
3. ✅ Откроется браузер с редактором
4. ✅ Можно использовать Graph Manager

## ЕСЛИ НЕ РАБОТАЕТ:
Установите:
- **Node.js** с https://nodejs.org/
- **Python 3** с https://www.python.org/

## ПОСЛЕ ЗАПУСКА:
Нажмите **"Graph Manager"** в редакторе

## ДЛЯ ОСТАНОВКИ:
Нажмите **Ctrl+C** в терминале
