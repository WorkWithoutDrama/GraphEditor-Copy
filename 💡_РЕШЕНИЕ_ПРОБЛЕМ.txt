# РЕШЕНИЕ ПРОБЛЕМ GRAPH EDITOR

## ПРОБЛЕМА: "Address already in use" (порт занят)

### РЕШЕНИЕ 1: Автоматическая очистка
```bash
./cleanup.command
```
Затем снова:
```bash
./RUN.command
```

### РЕШЕНИЕ 2: Вручную в терминале
```bash
# Посмотреть процессы на портах:
lsof -ti:3000
lsof -ti:5000

# Остановить все:
kill -9 $(lsof -ti:3000) 2>/dev/null
kill -9 $(lsof -ti:5000) 2>/dev/null
```

### РЕШЕНИЕ 3: Если не работает Node.js
Установите: https://nodejs.org/

### РЕШЕНИЕ 4: Если не работает Python
Установите: https://www.python.org/

## ПОСЛЕ ОЧИСТКИ:
```bash
./RUN.command
```

## ЕСЛИ ВСЁ РАБОТАЕТ:
1. Браузер откроется автоматически
2. Нажмите "Graph Manager" в редакторе
3. Отправьте описание системы

## ДЛЯ ОСТАНОВКИ:
Нажмите **Ctrl+C** в терминале где работает `RUN.command`
