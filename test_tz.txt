Техническое задание: Система управления задачами

Требуется разработать систему управления задачами со следующими функциями:

1. Создание задачи
   - Пользователь создает новую задачу
   - Задача содержит название, описание, приоритет, срок выполнения
   - Система сохраняет задачу в базе данных

2. Назначение задачи
   - Администратор назначает задачу исполнителю
   - Система отправляет уведомление исполнителю
   - Исполнитель подтверждает получение задачи

3. Выполнение задачи
   - Исполнитель изменяет статус задачи на "в работе"
   - Исполнитель добавляет комментарии к задаче
   - Исполнитель загружает прикрепленные файлы

4. Проверка задачи
   - Администратор проверяет выполненную задачу
   - Если задача выполнена правильно, статус меняется на "завершено"
   - Если есть замечания, задача возвращается на доработку

5. Отчетность
   - Система генерирует отчеты по выполненным задачам
   - Отчеты включают статистику по исполнителям
   - Отчеты экспортируются в формате PDF
