# ДЛЯ ЗАПУСКА GRAPH EDITOR:

## СПОСОБ 1 (простой):
**ДВАЖДЫ КЛИКНИТЕ НА `GO.command`**

## СПОСОБ 2 (если не работает):
1. Откройте Терминал
2. Введите:
   ```bash
   cd /путь/к/этой/папке
   chmod +x GO.command
   ./GO.command
   ```

## ЧТО БУДЕТ:
1. ✅ Запустится AI сервер
2. ✅ Запустится прокси  
3. ✅ Откроется браузер
4. ✅ Можно использовать Graph Manager

## ЕСЛИ ОШИБКА "порт занят":
Запустите сначала:
```bash
./cleanup.command
```
Или в терминале:
```bash
kill -9 $(lsof -ti:3000) 2>/dev/null
kill -9 $(lsof -ti:5000) 2>/dev/null
```

## ПОСЛЕ ЗАПУСКА:
1. Нажмите **"Graph Manager"** в редакторе
2. Отправьте **описание системы**
3. **AI создаст** графовую модель

## ДЛЯ ОСТАНОВКИ:
Нажмите **Ctrl+C** в терминале
