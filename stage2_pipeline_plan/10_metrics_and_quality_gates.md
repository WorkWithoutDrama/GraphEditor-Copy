# 10 — Metrics and Quality Gates

## Coverage
- % claims reviewed (not UNREVIEWED) by type
- # canonical claims by type
- average cluster size

## Quality indicators
- conflict group count
- reject rate
- low-confidence rate

## Regression checks
- re-run Stage‑2 on same inputs should produce no changes (idempotency)
- sample top merges with evidence to validate prompt behavior
